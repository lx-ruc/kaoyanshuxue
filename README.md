# 考研数学历年真题图片库

2009-2025 年考研数学一、数学二、数学三全部真题图片及答案，数据来源于 [真题墙](https://zhentiqiang.com/kaoyan/math)。

提供两种组织方式：

- **按年份分类**（`kaoyan_math_images/`）：适合按套卷刷题
- **按考点分类**（`kaoyan_math_by_topic/`）：适合按专题突破薄弱点

## 数据概览

| 项目 | 数量 |
|------|------|
| 试卷组 | 3（数学一 / 数学二 / 数学三） |
| 年份范围 | 2009 - 2025（共 17 年） |
| 题目总数 | 1,158 道 |
| 考点总数 | 68 个（数学一）、45 个（数学二）、57 个（数学三） |
| 图片总数 | 3,032 张 |
| 总大小 | ~1 GB |

### answers.json

每个试卷组目录下的 `answers.json` 包含所有题目的元数据：

```json
{
  "数学一_2025_Q01": {
    "group": "数学一",
    "paper": "2025",
    "index": 1,
    "question_id": 665,
    "type": 1,
    "answer": "B",
    "score": 5,
    "knowledge_point": "曲线的凹凸性、拐点及渐近线",
    "video_url": "https://www.bilibili.com/video/BV1Ut411p7k3?p=87&t=10"
  }
}
```

| 字段 | 说明 |
|------|------|
| `group` | 试卷组（数学一 / 数学二 / 数学三） |
| `paper` | 年份 |
| `index` | 题号（1-23） |
| `question_id` | 系统中的唯一题目 ID |
| `type` | `1` = 选择题，`2` = 填空/解答题 |
| `answer` | 选择题答案（如 "B"），非选择题为 `null` |
| `score` | 分值 |
| `knowledge_point` | 考点名称 |
| `video_url` | B 站名师讲解视频链接 |

### 三大科目考点一览

**高等数学（数学一，45 个考点）：**

| 章节 | 考点 |
|------|------|
| 函数、极限、连续 | 数列敛散性的判定、函数极限的计算、确定极限中的参数、无穷小量的比较、函数的连续性 |
| 一元函数微分学 | 导数与微分的概念/计算、导数的几何意义、单调性/极值/最值、凹凸性/拐点/渐近线、方程根、不等式证明、微分中值定理、泰勒公式 |
| 一元函数积分学 | 不定积分、定积分（概念/计算/应用）、变限积分、反常积分 |
| 多元函数微分学 | 偏导数、全微分、几何应用、方向导数和梯度、极值问题 |
| 多元函数积分学 | 重积分（概念/计算/应用/交换次序）、曲线积分（第一/二类）、曲面积分（第一/二类）、旋度 |
| 无穷级数 | 常数项级数、幂级数（收敛域/和函数/展开）、傅里叶级数 |
| 常微分方程 | 解的结构、可分离变量/齐次方程、一阶线性、常系数齐次/非齐次、微分方程应用 |

**线性代数（数学一，11 个考点）：**

行列式、矩阵（运算/伴随/秩）、向量（线性相关/线性表示/正交）、线性方程组、特征值与特征向量（含相似对角化）、二次型

**概率论与数理统计（数学一，12 个考点）：**

随机事件和概率、随机变量及其分布、多维随机变量（二维/边缘/条件）、数字特征（期望/方差/协方差/相关系数）、大数定律、数理统计基本概念、参数估计与假设检验（无偏性/矩估计/最大似然/区间估计/假设检验）

> 数学二不考概率论与数理统计，考点数较少（45 个）；数学三考点数介于两者之间（57 个）。

## 题目类型说明

| 题号 | 类型 | 答案形式 |
|------|------|----------|
| Q01 - Q08 | 选择题（type=1） | `answers.json` 中的字母（如 "B"），无答案图片 |
| Q09 - Q10 | 填空题（type=2） | 答案图片 + 解析图片 |
| Q11 - Q23 | 解答题（type=2） | 答案图片 + 解析图片 |

## 重复题说明

数学一、数学二、数学三在同一年份中存在部分重复题目（共用同一道题），这些题的 `question_id` 相同，图片内容一致。

## 脚本说明

| 脚本 | 用途 |
|------|------|
| `download_math_images.py` | 从真题墙 API 下载全部真题图片（按年份组织），提取答案数据 |
| `organize_by_topic.py` | 将按年份组织的图片复制一份，重新按考点分类组织 |

```bash
# 第一步：下载全部图片（按年份）
python3 download_math_images.py

# 第二步：生成考点分类视图（基于已下载的图片，不重复下载）
python3 organize_by_topic.py
```

两个脚本都支持断点续传，已存在的文件会自动跳过。
